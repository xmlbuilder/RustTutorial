# Synthetic Data

Synthetic Data(합성 데이터)는 실제 세계에서 수집된 데이터가 아니라,  
알고리즘이나 시뮬레이션을 통해 인위적으로 생성된 데이터입니다.  
즉, 현실 데이터를 모방하면서도 개인정보나 민감한 정보를 포함하지 않는 안전한 대체 데이터.

## 📌 Synthetic Data의 정의
- 인위적으로 생성된 데이터: 실제 사건이나 센서에서 나온 것이 아니라, 수학적 모델·통계적 기법·AI(예: GAN, Diffusion, 시뮬레이션)로 만들어짐.
- 실제 데이터와 유사한 분포: 원본 데이터의 통계적 특성과 패턴을 모방해 학습·테스트에 활용 가능.
- 다양한 형태:
  - Tabular: 표 형식 데이터 (예: 금융 거래 기록)
  - Text: 문장, 대화 데이터 (NLP 모델 학습용)
  - Image/Video: 합성 이미지, 가상 환경 시뮬레이션 (컴퓨터 비전용)

## 🎯 Synthetic Data가 필요한 이유
- 데이터 부족 문제 해결: 실제 데이터가 적거나 수집이 어려운 경우 대체 가능
- 프라이버시 보호: 민감한 개인정보를 포함하지 않으면서도 원본 데이터와 유사한 학습 효과 제공
- 모델 성능 향상: 데이터 증강(data augmentation)처럼 모델의 일반화 성능을 높임
- 테스트 및 시뮬레이션: 실제 환경에서 얻기 힘든 극단적 상황(예: 금융 사기, 드론 충돌)을 합성 데이터로 재현 가능

## 📊 활용 사례
- 의료 분야: 환자 개인정보를 보호하면서 AI 모델 학습용 데이터 생성
- 금융 분야: 실제로는 드문 사기 거래 패턴을 합성해 탐지 모델 훈련
- 자율주행: 가상 환경에서 다양한 도로·날씨·사고 상황을 합성해 차량 AI 학습
- 국방/항공: 시뮬레이션 기반 합성 데이터로 레이더·센서 시스템 테스트

## 📌 요약
- 👉 Synthetic Data는 현실 데이터를 모방해 인위적으로 생성된 데이터로, 데이터 부족·프라이버시 문제를 해결하고 AI 모델 학습과 테스트에 널리 활용됩니다.
