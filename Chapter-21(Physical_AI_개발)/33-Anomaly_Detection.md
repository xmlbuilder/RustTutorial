# Anomaly Detection
Anomaly Detection(이상 탐지)은 데이터나 시스템에서 정상적인 패턴과 크게 벗어나는 드문 사건이나 관측값을 찾아내는 기술입니다.    
흔히 Outlier Detection 또는 Novelty Detection이라고도 불리며, 정상적인 행동과 비교했을 때 비정상적인 데이터 포인트를 식별하는 것이 목적.

## 📌 정의
- 일반적 의미: 데이터 집합에서 정상적인 분포나 패턴과 맞지 않는 값, 이벤트, 관측치를 탐지하는 것.
- 다른 이름: Outlier Detection, Novelty Detection.
- 핵심 아이디어: 정상 행동을 모델링한 뒤, 그와 크게 다른 데이터가 들어오면 "이상(anomaly)"으로 분류.

## 📊 주요 기법
- Supervised (지도 학습)
  - 정상/비정상 데이터가 라벨링된 경우, 분류기를 학습시켜 탐지.
  - 하지만 실제로는 라벨링된 이상 데이터가 부족해 잘 쓰이지 않음.
- Semi-supervised (준지도 학습)
  - 정상 데이터만 라벨링된 경우, 정상 행동 모델을 학습 후 새로운 데이터가 그 모델과 얼마나 다른지 평가.
- Unsupervised (비지도 학습)
  - 라벨이 없는 경우, 데이터의 분포나 군집을 분석해 이상치 탐지.
  - 가장 널리 쓰이는 방식 (예: k-NN, Isolation Forest, Local Outlier Factor).

## 🎯 활용 분야
- 사이버 보안: 네트워크 침입, 비정상 트래픽 탐지
- 금융: 신용카드 사기 거래 탐지
- 제조업: 센서 데이터 기반 결함 탐지
- 의료: 환자 데이터에서 이상 징후 탐지
- 통계/데이터 과학: 평균·분산을 왜곡하는 이상치 제거

## 📌 요약
- 👉 Anomaly Detection은 정상 패턴과 크게 벗어나는 데이터나 이벤트를 탐지하는 기술로, 보안·금융·의료 등 다양한 분야에서 활용됩니다.
- 👉 라벨링 여부에 따라 Supervised, Semi-supervised, Unsupervised 방식으로 나뉘며, 실제 현업에서는 비지도 학습 기반 기법이 가장 많이 쓰입니다.
