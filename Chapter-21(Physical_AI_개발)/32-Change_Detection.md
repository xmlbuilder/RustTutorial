# Change Detection

## 📌 Change Detection의 정의
- 통계적 관점: 시계열 데이터나 확률 과정에서 **분포가 변하는 시점(change point)** 을 찾아내는 것.
- 공간적 관점(GIS/원격탐사): 위성 이미지, 항공사진 등을 여러 시점에서 비교해 지표면, 도시, 환경의 변화를 탐지하는 것.
- 일반적 의미: 객체, 시스템, 환경의 상태가 달라졌는지 확인하는 모든 분석 기법을 포함.

## 📊 주요 방법
- 통계적 방법: 평균, 분산, 상관관계가 변하는 시점을 찾음 (예: Page test, CUSUM).
- 이미지 기반 방법:
  - Image differencing (두 시점 이미지 차이 계산)
  - Post-classification comparison (분류 후 결과 비교)
  - Change vector analysis (특징 벡터 변화 분석)
- 머신러닝 기반 방법: 딥러닝 모델을 활용해 변화 영역 자동 탐지.

## 🎯 활용 분야
- 환경 모니터링: 산림 벌채, 사막화, 수자원 변화 추적
- 도시 개발: 건물 신축, 도로 확장, 토지 이용 변화
- 재난 관리: 홍수, 지진, 화재 등 피해 지역 탐지
- 보안/산업: 네트워크 이상 탐지, 제조 공정 변화 감지

## 📌 요약
- 👉 Change Detection은 시간에 따른 데이터나 환경의 변화를 탐지하는 기술로, 통계·GIS·머신러닝 등 다양한 방법을 활용합니다.
- 👉 환경 과학, 도시 계획, 재난 대응, 산업 모니터링 등 광범위하게 적용됩니다.
