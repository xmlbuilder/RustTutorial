# Edge AI 공부 시작
👉 Edge AI를 공부하거나 연구를 시작하려면, 기본적인 AI/ML 지식 위에 **엣지 환경(저전력·실시간·임베디드)** 에  
맞는 기술을 배우는 것이 핵심입니다.  
클라우드 중심 AI와 달리, Edge AI는 디바이스 자체에서 모델을 실행하기 때문에 최적화·경량화·하드웨어 이해가 중요.

## 📌 학습 시작 단계
### 1. 기본기 다지기
- 머신러닝/딥러닝 기초: Python, PyTorch, TensorFlow로 모델 학습 경험 쌓기
- 임베디드 시스템 이해: 마이크로컨트롤러(MCU), GPU, FPGA 같은 하드웨어 구조 학습
- Edge Computing 개념: 클라우드 vs 엣지 차이, 지연(latency)·프라이버시·전력 제약 이해
### 2. Edge AI 전용 학습 자료
- Microsoft Edge AI for Beginners
  - 무료 오픈소스 커리큘럼, GitHub에서 제공
  - 소형 언어 모델(SLM), Llama.cpp, Olive, OpenVINO 같은 툴로 로컬 추론 실습
- Edge AI Foundation
  - 글로벌 커뮤니티, 인증·배지 시스템, 오픈소스 데이터셋 제공
- 실습 환경
  - Raspberry Pi, NVIDIA Jetson, Arduino 같은 엣지 디바이스에서 직접 모델 배포
3. 연구 방향
  - 모델 경량화: 양자화(Quantization), 프루닝(Pruning), 지식 증류(Knowledge Distillation)
  - 실시간 최적화: 저지연 추론, 메모리 효율화
  - 응용 분야 연구: 자율주행, 스마트 팩토리, 헬스케어, 국방 등

## 📊 추천 학습 로드맵
- AI 기초: Python ML/DL → CNN, RNN, Transformer 이해
- Edge 환경 이해: IoT, 임베디드 시스템, 하드웨어 가속기 학습
- Edge AI 실습: Jetson Nano, Raspberry Pi에서 모델 배포
- 최적화 연구: 모델 압축·경량화 기법 적용
- 프로젝트 진행: 이미지 분류, 음성 인식, 센서 데이터 이상 탐지 등 Edge AI 응용

## 📌 요약
- 👉 Edge AI를 시작하려면 AI 기본기 + 임베디드 시스템 이해 + 모델 최적화 기술을 함께 공부해야 합니다.
- 👉 Microsoft의 Edge AI for Beginners 같은 무료 커리큘럼과 Edge AI Foundation의 글로벌 커뮤니티가 좋은 출발점이 됩니다.
