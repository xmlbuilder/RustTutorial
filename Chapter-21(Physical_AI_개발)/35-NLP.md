# NLP
NLP(Natural Language Processing, 자연어 처리)는 컴퓨터가 인간의 언어를 이해하고, 해석하고, 생성할 수 있도록 하는 인공지능(AI) 분야입니다.  
즉, 사람이 쓰는 텍스트나 음성을 기계가 처리할 수 있게 만드는 기술.

## 📌 정의
- **자연어 처리(NLP)** 는 컴퓨터 과학, 인공지능, 언어학의 교차 분야로, 기계가 인간 언어를 다룰 수 있도록 하는 기술.
- 핵심 목표: 인간 언어를 컴퓨터가 인식 → 의미를 이해 → 적절한 응답을 생성.
- 방법론: 규칙 기반 언어 모델링, 통계적 모델링, 머신러닝, 딥러닝 기법을 결합.

## 📊 주요 기능
- 텍스트 처리: 문장 분류, 감성 분석, 요약, 번역
- 음성 처리: 음성 인식, 음성 합성
- 자연어 이해(NLU): 질문에 답하기, 의미 추출
- 자연어 생성(NLG): 대화, 글쓰기, 보고서 자동 생성

## 🎯 활용 사례
- 검색 엔진: 사용자의 질의어를 이해해 적절한 결과 제공
- 챗봇/디지털 비서: Siri, Alexa, Copilot 같은 대화형 AI
- 자동 번역: Google Translate, DeepL
- 문서 처리: 자동 요약, 정보 추출, 분류
- 소셜 미디어 분석: 감성 분석, 트렌드 탐지

## 📌 요약
- 👉 NLP는 컴퓨터가 인간 언어를 이해하고 생성하는 기술로, 검색 엔진·챗봇·번역·음성 인식 등 다양한 분야에서 활용됩니다.
- 👉 현대 NLP는 딥러닝 기반 대규모 언어 모델(LLM) 덕분에 비약적으로 발전했으며, 오늘날의 생성형 AI의 핵심 기반 기술입니다.
