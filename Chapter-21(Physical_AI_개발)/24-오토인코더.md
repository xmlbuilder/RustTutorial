## 📌 오토인코더 기본 개념

- 오토인코더는 **입력 데이터를 압축(인코딩)** 했다가 다시 **복원(디코딩)** 하는 신경망입니다.
- 정상적인 데이터(예: 배경 이미지, 정상 신호)를 학습하면, 그 데이터는 잘 복원할 수 있습니다.
- 하지만 학습하지 않은 **이상 데이터(표적, 비정상 패턴)** 는 복원이 잘 안 되고, **재구성 오차(reconstruction error)** 가 크게 나타납니다.

## 📊 **재구성 오차로 표적 탐지** 의 의미
- 정상 데이터로 오토인코더 학습
  - 예: 위성 영상에서 **배경** 만 학습시킴.
- 새로운 입력을 오토인코더에 통과
  - 정상적인 배경은 잘 복원됨 → 오차가 작음.
  - 표적(탐지 대상, 이상 객체)은 복원이 잘 안 됨 → 오차가 큼.
- 재구성 오차를 기준으로 탐지
  - 오차가 큰 영역을 **표적** 으로 판단.

## 📌 활용 예시
- 위성/항공 영상 표적 탐지: 군사 표적, 차량, 건물 등 비정상 객체 탐지
- 의료 영상: 정상 조직만 학습 → 병변(종양 등)은 재구성 오차로 탐지
- 산업 검사: 정상 제품만 학습 → 결함 부위는 오차로 탐지
- 사이버 보안: 정상 네트워크 트래픽 학습 → 공격 패턴은 오차로 탐지

## 📌 요약
- **오토인코더 재구성 오차로 표적 탐지** 란,
- 👉 정상 데이터만 학습한 오토인코더가 비정상 데이터를 잘 복원하지 못하는 성질을 이용해, 오차가 큰 부분을 ‘표적’으로 탐지하는 방법입니다.
