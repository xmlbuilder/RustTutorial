# Few-shot Learning
**Few-shot Learning(소수 샷 학습)** 은 머신러닝과 딥러닝에서 아주 적은 수의 학습 데이터만으로도  
새로운 작업을 수행할 수 있도록 하는 학습 방법입니다.

## 📌 정의
- 전통적 학습: 수천~수만 개의 데이터가 필요.
- Few-shot Learning: 단 몇 개(예: 1~50개 정도)의 예시만으로도 모델이 새로운 태스크를 학습.
- 핵심 아이디어: 이미 대규모 데이터로 학습된 **사전 학습 모델(pre-trained model)** 의 일반화 능력을 활용해, 소량의 데이터로 빠르게 적응(fine-tuning)하는 것.

## 📊 주요 접근 방식
- Meta-learning (메타 학습)
  - "학습하는 방법을 학습" → 모델이 새로운 태스크에 빠르게 적응할 수 있도록 설계.
  - 예: MAML(Model-Agnostic Meta-Learning).
- Metric-based Learning (거리 기반 학습)
  - 새로운 데이터와 기존 데이터 간의 유사도를 측정해 분류.
  - 예: Prototypical Networks, Siamese Networks.
- Fine-tuning (미세 조정)
  - 대규모 데이터로 학습된 모델을 가져와, 소량의 데이터로 일부 레이어만 조정.
  - 예: GPT, BERT 같은 언어 모델을 특정 도메인에 맞게 소량 데이터로 학습.

## 🎯 활용 분야
- 자연어 처리(NLP): 소량의 예시 문장만으로 새로운 태스크(예: 감성 분석, 질의응답) 수행.
- 컴퓨터 비전: 몇 장의 이미지로 새로운 클래스 인식.
- 의료 분야: 환자 데이터가 적을 때 질병 분류 모델 학습.
- 보안/산업: 드문 이상 패턴을 소량 데이터로 학습해 탐지.

## 📌 요약
- 👉 Few-shot Learning은 소량의 데이터만으로도 새로운 작업을 학습할 수 있는 방법으로,  
  대규모 사전 학습 모델과 메타 학습 기법을 활용합니다.
- 👉 데이터가 부족한 환경(의료, 보안, 희귀 이벤트 탐지)에서 특히 강력한 도구로 쓰입니다.
