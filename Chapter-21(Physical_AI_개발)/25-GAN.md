# GAN
GAN은 **Generative Adversarial Network(적대적 생성 신경망)** 의 약자로, 새로운 데이터를 만들어내는 데 특화된 딥러닝 모델입니다.  
두 개의 신경망(생성자와 판별자)이 서로 경쟁하면서 점점 더 사실적인 데이터를 생성할 수 있게 됩니다.

## 📌 GAN의 핵심 구조
- 생성자(Generator)
  - 무작위 입력(노이즈)을 받아서 가짜 데이터를 생성합니다.
  - 목표: 판별자를 속여서 **진짜처럼 보이는** 데이터를 만들어내기.
- 판별자(Discriminator)
  - 입력된 데이터가 진짜(학습 데이터)인지, 가짜(생성자가 만든 데이터)인지 판별합니다.
  - 목표: 가짜를 잘 구별해내기.
- 학습 과정
  - 생성자는 판별자를 속이려고 노력하고, 판별자는 속지 않으려고 노력합니다.
  - 이 경쟁(적대적 학습)을 통해 생성자는 점점 더 사실적인 데이터를 만들어내고, 판별자는 점점 더 정교하게 구별합니다.

## 📊 GAN의 특징
- 비지도 학습 기반: 레이블 없이도 데이터의 분포를 학습할 수 있음.
- 창의적 생성 능력: 사진, 그림, 음악, 텍스트 등 다양한 형태의 데이터를 새롭게 만들어낼 수 있음.
- 진화적 경쟁 구조: 생물학의 **군비 경쟁(arms race)** 과 유사하게 두 네트워크가 서로 발전함.

## 🎯 GAN의 활용 분야
- 이미지 생성: 사람 얼굴, 풍경, 예술 작품 등 사실적인 이미지 합성
- 영상/음성 합성: 딥페이크, 음성 합성, 음악 생성
- 데이터 증강: 부족한 학습 데이터를 보완하기 위해 새로운 샘플 생성
- 의료 영상: 희귀 질환 데이터 증강, 영상 복원
- 스타일 변환: 사진을 그림처럼 바꾸는 등 이미지 스타일 변환

## 📌 요약
- GAN은 생성자와 판별자가 경쟁하는 구조를 통해, 기존 데이터와 구분하기 어려울 정도로 사실적인 새로운 데이터를 만들어내는 딥러닝 모델입니다.
- 오늘날 AI의 창의적 생성 능력(이미지 합성, 딥페이크, 예술적 창작 등)의 핵심 기술 중 하나입니다.
