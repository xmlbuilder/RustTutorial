# Deep Learning

Deep Learning은 인공 신경망을 여러 층으로 깊게 쌓아 복잡한 데이터 패턴을 자동으로 학습하는 인공지능 기법입니다.  
머신러닝의 한 분야로, 이미지·음성·텍스트 같은 비정형 데이터를 처리하는 데 특히 강력한 성능을 발휘합니다.

## 📌 Deep Learning의 핵심 개념
- 다층 신경망(Multi-layer Neural Network)
  - 입력층 → 여러 개의 은닉층 → 출력층 구조로 이루어져 있으며, 각 층은 데이터를 점점 더 추상적인 표현으로 변환합니다.
- 비선형 변환
  - 각 뉴런은 단순한 선형 연산뿐 아니라 ReLU, Sigmoid 같은 비선형 활성화 함수를 사용해 복잡한 관계를 학습합니다.
- 자동 특징 추출
  - 전통적 머신러닝은 사람이 특징(feature)을 직접 설계해야 했지만, 딥러닝은 데이터를 통해 특징을 자동으로 학습합니다.
- 대규모 데이터와 GPU 필요
  - 딥러닝은 수많은 파라미터를 학습하기 때문에 **방대한 데이터와 고성능 연산 자원(GPU/TPU)** 이 필요합니다.

## 📊 머신러닝 vs 딥러닝

| 구분        | 머신러닝                                   | 딥러닝                                   |
|-------------|--------------------------------------------|------------------------------------------|
| 특징 추출   | 사람이 직접 설계                           | 데이터로부터 자동 학습                   |
| 데이터 요구 | 적은 데이터로도 가능                       | 대규모 데이터 필요                       |
| 모델 구조   | 통계적 알고리즘 기반                       | 다층 신경망(Deep Neural Network) 기반     |
| 해석 용이성 | 비교적 쉬움                                | 블랙박스 성격 강함                       |
| 성능        | 단순 문제에 적합                           | 복잡한 문제(이미지, 음성, NLP)에 강력    |

## 🎯 Deep Learning의 주요 응용
- 컴퓨터 비전: 이미지 분류, 객체 탐지, 얼굴 인식 (CNN 기반)
- 자연어 처리(NLP): 번역, 챗봇, 감정 분석 (RNN, Transformer 기반)
- 음성 인식: 스마트 스피커, 자동 자막 생성
- 생성 모델: GAN, Diffusion 모델을 통한 이미지·영상·음악 생성
- 의료·과학: MRI 분석, 신약 개발, 기후 예측

## 📌 요약
- 👉 Deep Learning은 다층 신경망을 통해 데이터의 복잡한 패턴을 자동으로 학습하는 기술로, 오늘날 AI의 핵심 엔진입니다.
- 이미지·음성·텍스트 같은 다양한 분야에서 인간 수준을 뛰어넘는 성능을 보여주며, 머신러닝보다 더 큰 데이터와 연산 자원을 필요로 합니다.
