# CNN
CNN은 ‘Convolutional Neural Network(합성곱 신경망)’의 약자로, 이미지·영상 같은 시각 데이터를 처리하는 데 특화된 딥러닝 모델입니다.  
주로 컴퓨터 비전 분야에서 사용되며, 패턴 인식·분류·검출에 강력한 성능을 발휘합니다.

## 📌 CNN의 핵심 개념
- 합성곱(Convolution) 연산
  - 작은 필터(커널)를 이미지 전체에 슬라이딩하며 특징을 추출합니다.
    - 가장자리, 질감, 모양 같은 지역적 패턴을 자동으로 학습.
- 계층적 특징 학습
  - 하위 층: 선, 모서리 같은 단순 패턴
  - 상위 층: 얼굴, 물체 같은 복잡한 구조
- CNN은 층을 거듭할수록 더 추상적인 특징을 학습합니다.
  - 파라미터 공유와 지역 연결
- 필터를 공유하기 때문에 일반적인 완전연결 신경망보다 학습 파라미터 수가 적고 효율적입니다.

## 📐 CNN의 기본 구조
- 입력층(Input Layer): 이미지(예: 32×32×3 RGB)
- 합성곱층(Convolution Layer): 필터로 특징 추출
- 풀링층(Pooling Layer): 크기 축소, 불변성 확보 (예: Max Pooling)
- 완전연결층(Fully Connected Layer): 추출된 특징으로 최종 분류
- 출력층(Output Layer): 예측 결과 (예: 고양이/개 분류)

## 📊 CNN의 활용 분야
- 이미지 분류: 손글씨 인식(MNIST), ImageNet 대규모 이미지 분류
- 객체 검출: 얼굴 인식, 자율주행 자동차의 보행자 검출
- 의료 영상 분석: CT/MRI에서 병변 탐지
- 자연어 처리(NLP): 문장 분류, 감정 분석에도 응용 가능
- 추천 시스템, 금융 시계열 분석 등 다양한 데이터 처리.

## 📌 요약
- CNN은 이미지와 같은 격자형 데이터에서 특징을 자동으로 추출하고 학습하는 딥러닝 모델입니다.  
- 합성곱과 풀링을 통해 효율적으로 패턴을 인식하며, 오늘날 컴퓨터 비전과 AI의 핵심 기술로 자리잡고 있습니다.
